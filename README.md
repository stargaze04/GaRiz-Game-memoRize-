# GaRiz-Game-memoRize-
Gariz atau Game Memorize adalah permainan uji memori klasik (dikenal juga sebagai "Concentration") yang disajikan dalam antarmuka berbasis teks yang bersih di lingkungan terminal. Tujuan utama permainan ini adalah mencocokkan semua pasangan kartu identik yang tersembunyi di papan permainan grid 4 x 4 dengan upaya melakukan percobaan sesedikit mungkin.
